# Hello World

This is a simple "Hello World" website project.  

It is designed for beginners to learn basic **HTML** and **CSS**.  

## How to Use

1. Download or clone this repository.
2. Open the `index.html` file in your browser.
3. You should see a page displaying "Sannu, Duniya!" and a short paragraph.

## Files

- `index.html` — the main HTML page
- `style.css` — the styling for the page
- `README.md` — this file explaining the project

## Goal

The goal of this project is to help beginners start their journey with web development.
